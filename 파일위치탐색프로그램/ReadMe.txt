==================================== 사용방법 ============================================

1. 파일 확장자 선택창에 들어가서 원하는 확장자를 선택한다.
2. 파일 확장자 선택하고 Extension Save 버튼을 클릭하여 원하는 파일 확장자 저장한다. (아무것도 체크가 안되어 있다면 모든 확장자 파일들의 위치를 알려준다)
3. 파일 찾기 창에 들어가 원하는 폴더를 선택하고 가억나는 파일 이름을 적는다. (파일 이름을 안적으면 확장자에 해당하는 모든 이름의 파일을 가지고 온다) 
4. 파일 검색 버튼을 클릭한다. (문서, 바탕화면등의 폴더를 선택시 시간이 다소 걸릴 수 있으니 기다려주세요)
5. 리스트에 파일 경로들이 띄어지고 "파일들을 다운로드 하시겠습니까?" 라는 문구가 뜨고 "예(Y)" 버튼을 누르면 바탕화면에 폴더가 생성되고 그안에 파일들이 저장된다.
